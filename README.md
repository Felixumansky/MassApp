# MassApp 💪 — מעקב עלייה למסה

אפליקציית ווב מותאמת לטלפון, בעברית מלאה (RTL): מעקב קלוריות, ארוחות, מאקרו, מים ומשקל.

- **פרונט:** React + Vite + Tailwind v4 (התקייה הזו)
- **באק:** Node.js + Express — [`../MassAPI`](../MassAPI)
- **DB:** Supabase
- **איפיון מלא:** [SPEC.md](SPEC.md)

## הרצה

1. **הגדר את ה-Backend** — עקוב אחרי ההוראות ב-`../MassAPI/README.md` (יצירת Supabase, `.env`, `schema.sql`).
2. הרץ את ה-Backend:
   ```bash
   cd ../MassAPI && npm run dev
   ```
3. הרץ את הפרונט (טרמינל שני):
   ```bash
   npm install
   npm run dev
   ```
4. פתח בטלפון/דפדפן: `http://localhost:5173` (בפיתוח יש proxy אוטומטי ל-API).

## מסכים

| מסך | מה יש בו |
|------|-----------|
| 🏠 היום | טבעת קלוריות מונפשת, פסי מאקרו, מד מים עם כפתורים מהירים, ארוחות לפי סוג |
| ➕ הוספה | קטלוג מאכלים ישראלי עם חישוב אוטומטי לפי כמות, או הזנה ידנית |
